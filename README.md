# Тестовое задание Selsup
## Для того, чтобы запустить проект, необходимо написать:

Копирование проекта
```console
git clone https://github.com/horspid/selsup-test.git
```
Установка зависимостей
```console
npm i
```

Запуск проекта
```console
npm start
```
